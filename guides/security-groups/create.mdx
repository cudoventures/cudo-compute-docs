titleDir: Security groups

tag: Guide
---

Open a project and select **Security groups**.

Click **Create security group** and complete:

- Security group ID - unique name
- Data center - deployment location
- Description - optional text

Click **Create security group**.

<Note>
New security groups start with no rules. Add rules to allow traffic.
</Note>

## Deploy a virtual machine with a security group

When creating a virtual machine choose the security group in the Security groups step.
