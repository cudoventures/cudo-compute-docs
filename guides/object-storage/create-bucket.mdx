## Create a bucket
Open **Object storage > Buckets** and click **Create a bucket**. Provide:

- Data center - where the bucket is created
- Name - unique bucket name

Click **Create bucket**.

## Create a folder
Inside the bucket click **Create folder**, enter a name and confirm.

## Upload files
Open a folder and click **Select files** to upload.
