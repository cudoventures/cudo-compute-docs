A bare metal machine is a physical server that is not virtualized. This means that the entire machine is dedicated to a single user or application, providing maximum performance and control.

## Features

- Whole machine dedicated to a single user (up to 8 GPUs per machine)
- No virtualization overhead
- Full control over hardware and software
- SSH access to the machine
- Range of operating systems available (including Cloud init configuration support)
- Project based permissions

### Operating systems

CUDO Compute offers a range of operating systems for bare metal machines, including:
- Ubuntu 22.04 (ubuntu/jammy)
- Ubuntu 24.04 (ubuntu/noble)
