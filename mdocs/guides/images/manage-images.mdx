## View images

On the Images page see all private and public images available to you:

- Private: ID, description, data center, size
- Public: name, ID, size

<Note>
Public images are available at all data centers.
</Note>

## Delete a private image

Click **Delete** on the image you want to remove.

<Note variant="warning">
You can only delete a private image if no virtual machines are using it.
</Note>

## Deploy using an image

When creating a virtual machine choose the private image from the image selector.
