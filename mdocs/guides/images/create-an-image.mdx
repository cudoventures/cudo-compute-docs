---

<Note>
To create a private image, you first need a virtual machine. See Create a virtual machine.
</Note>

Navigate to the virtual machine details page of the VM whose disk you want to image. In the Disks table open the action menu for the boot disk and select **Create image from disk**.

<Note>
Private images are charged at the storage rate for their data center.
</Note>

Provide:

- Image ID - unique identifier
- Description - optional description

Click **Create image**. After creation you can deploy a new VM using this image.
