## SSH

From the virtual machine list use **⋮ > SSH** or from the virtual machine details page use the quick action to copy the SSH command and run it locally.

<Note>
See tutorial on generating SSH keys and adding them to your virtual machine.
</Note>

## VNC

From the virtual machine list choose **⋮ > Start VNC session** or use the quick action on the details page.

<Note>
Set the root/administrator password first to access via VNC.
</Note>

Use quick buttons in the VNC session window to show the on-screen keyboard or send Ctrl+Alt+Delete.
