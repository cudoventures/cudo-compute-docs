---
title: Create a disk
titleDir: Disks
description: Create detachable disks on CUDO Compute
tag: Guide
---

Navigate to the project in which you want to create a disk and open the **Disks** page.

Click **Create a disk**. A form appears with the following fields:

- Name - identifier for the disk
- Size (GiB) - capacity of the disk
- Data center - location (only VMs in this data center can attach this disk)

<Note>
You can only attach this disk to virtual machines within the selected data center.
</Note>

Click **Create disk** to provision it.
