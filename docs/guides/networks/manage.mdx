## View networks

Open **Networks** to list all deployed networks with ID, gateway IP, IP range and state.

Click a network to open its details page showing the same information plus connected virtual machines.

## Delete a network

From the network details page open **Settings** and choose **Delete network**.

<Note variant="warning">
You can only delete a network if no virtual machines are connected to it.
</Note>

## Deploy a virtual machine to your network

When creating a virtual machine select the network during configuration.
