## Create a user

In a project open **Object storage > Users** and click **Create a user**. Provide:

- Data center - where the user is created
- Name - user identifier

Click **Create user**.

## View user keys
Click the user ID to view existing S3 keys or generate a new key with **Generate a key**.

## Edit user settings
On the user page open **Settings** to adjust access rights or delete the user.
