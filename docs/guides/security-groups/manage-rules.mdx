## Add a rule

Open a security group and click **Add rule** then specify:

- Direction - inbound or outbound
- Protocol - All, TCP, UDP, ICMP, ICMPv6, IPsec
- IP range - CIDR range to permit
- Ports - port or range (TCP/UDP only)
- ICMP type - type code (ICMP/ICMPv6 only)

Click **Save**.

## View rules

Select the security group to list all configured rules.
