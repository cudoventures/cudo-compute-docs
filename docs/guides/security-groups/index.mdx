---
title: Security groups
description: What are security groups & how can I use them on CUDO Compute?
tag: Concept
---

A security group is a set of firewall rules controlling traffic for your instances. After creating a security group in a data center you can define inbound and outbound rules. Security groups can be configured to manage public traffic and traffic within IP ranges of your private networks.

## Pricing

Security groups are free.
